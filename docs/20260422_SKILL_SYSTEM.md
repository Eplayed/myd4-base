## 技能分支(mods)结构分析

从 `skills_zhCN.json` 分析：

### 分支类型
- **强化 (Enhanced)**: modId=1，蓝色标记
- **初阶 (Prime)**: modId=0，紫色标记  
- **终极 (Supreme)**: 红色标记
- **职业特定变体**: 其他modId，绿色标记

### 示例
```
突刺打击 (Paladin):
  - 强化突刺打击 (modId: ?)
  - 战斗突刺打击
  - 争斗突刺打击
```
