# myd4-base 开发计划

> 暗黑破坏神4 数据查看器 & 构筑模拟器

仓库地址：https://github.com/Eplayed/myd4-base

---

## 一、已完成功能

### 数据查看器（viewer/）

| 标签页 | 功能 | 状态 |
|--------|------|------|
| 暗金 | 暗金装备卡片浏览，支持职业/装备/BOSS筛选，图片展示，点击弹窗详情 | ✅ |
| 威能 | 威能卡片浏览，支持职业/来源/类型筛选，能量法典与传奇威能分类 | ✅ |
| 词缀 | 左右双栏表格（普通词缀/回火词缀），支持职业+装备筛选，可展开查看effectList各档数值 | ✅ |
| 技能 | 7职业技能卡片浏览，72px图标，描述颜色格式化（标签橙/数值青/数字蓝），支持职业/类型/标签筛选 | ✅ |
| 符文 | 符文数据浏览 | ✅ |
| 宝石 | 宝石数据浏览 | ✅ |
| 药剂 | 药剂数据浏览 | ✅ |
| 召唤 | 召唤数据浏览 | ✅ |
| 构筑BD | 构筑列表页（占位） | ✅ |
| 模拟器 | 构筑模拟器简化版 - 装备系统 | ✅ |

### 数据文件（data/）

- `uniqueItem_zhCN.json` — 暗金装备
- `aspect_zhCN.json` — 威能
- `affix_zhCN.json` / `affix_d2core.json` — 词缀（双数据源）
- `skills_zhCN.json` — 技能（450条，7职业）
- `rune_zhCN.json` — 符文
- `gem_zhCN.json` — 宝石
- `elixir_zhCN.json` — 药剂
- `summon_zhCN.json` — 召唤

### 构筑模拟器 - 装备系统（已完成）

- 10个装备槽：头盔/胸甲/手套/裤子/靴子/主手/副手/护符/戒指1/戒指2
- 点击装备槽弹出物品选择模态框，从暗金装备中筛选对应部位
- 职业选择下拉框
- 右侧属性统计面板（生命/护甲/力量/敏捷/智力/意力/暴击等）
- 重置功能
